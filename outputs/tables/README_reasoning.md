# Reasoning Tables

This directory contains CSVs generated by the `reasoning-report` command.

Files:
- `reasoning_summary_by_source.csv`: aggregate metrics by `source_type` (LLM vs human vs book). Includes mean engine delta, mean specificity, recoverability rates, and average length.
- `per_puzzle_reasoning_comparison.csv`: one row per puzzle with side-by-side LLM and human fields (move, engine delta, specificity, recoverability).
- `explanation_features.csv`: per-explanation feature table (square mentions, move mentions, tactical keywords, generic phrases, line depth, specificity score).

All tables are derived from `data/processed/recoverability_results.jsonl`.
